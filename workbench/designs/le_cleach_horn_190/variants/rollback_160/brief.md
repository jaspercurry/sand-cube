# 160° rollback variant — user brief

## User request

> How hard would it be to increase the rollback? I think right now it's got like a 140° angle for that rollback. What would a 160° look like? Would that be hard for you to do?

## Working interpretation

“Rollback” is the terminal wall angle measured from the horn axis, matching the
existing `exit_angle_deg` parameter. The current 140° horn remains the baseline;
this iteration creates a separate 160° comparison candidate.
