# Enclosure V1 release area

This directory is the clean, final V1 boundary for the enclosure. Released
parts are immutable: development and experiments happen elsewhere, and a
future revision receives a new release version instead of modifying V1.

## Parts

- `front_baffle/` — released and currently being printed.
- `bucket/` — reserved for the separately reviewed V1 bucket release.

The bucket release must remain a sibling of `front_baffle/`. It must not copy,
regenerate, or modify the released baffle.
