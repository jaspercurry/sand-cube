# Single-land corner-fastener closure

This isolated experiment inherits the complete simplified printable closure and
changes only the removable-front joint:

- one gasket land and one matching bed-grown bucket ramp generated from the
  same perimeter;
- exact circular upper-corner arcs around the hidden sand-fill mouths;
- top and bottom M4 axes moved to the outer land/socket corner;
- compact bucket supports rooted into both the land underside and vertical
  socket wall;
- open baffle reliefs for straight drop-on assembly; and
- deeper, side-loaded square-nut cassettes with a five-millimetre nominal
  load-bearing shoulder.

Generated STEP files, diagnostics, and static viewers are written to the
matching directory under `build/`.
