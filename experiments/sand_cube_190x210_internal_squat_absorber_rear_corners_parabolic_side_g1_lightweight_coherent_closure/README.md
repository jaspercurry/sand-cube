# Lightweight coherent closure

This isolated sibling preserves the complete exterior, nested seam, driver
collar, recessed top/bottom screws, port, absorber, tower, horn, GX16, and
mounting system from the systemic-joint baseline.

It replaces the internal closure and brace implementation with four coherent
systems:

- a continuous 6.75 mm baffle gasket ring, the preserved driver collar, and
  four broad tapered bed-contact bridges instead of a full 3 mm diaphragm;
- one bucket wall-stack cap, gasket land, and rear-grown ramp sharing the same
  exact perimeter, with only the two intentional sand-fill passages;
- two simple bed-grown M4 nut blocks, each with the shortest straight loading
  slot from the printer-bed gasket/collar face in the screw bore's center plane
  and a coaxial terminal GB6170 hex seat; and
- directly constructed transverse, longitudinal, front-root, rear-cradle, and
  tower-cradle braces cut only by named hardware and airway envelopes.

The installed bucket and baffle are included in the authoritative cabinet
volume. Generated files are written only to the matching directory under
`build/`.

Primary printable STEP files:

- `centered_captive_nut_bucket.step` — bucket, printed rear face down;
- `centered_captive_nut_baffle.step` — removable front baffle; and
- `centered_captive_nut_baffle_finned_variant.step` — optional four-diagonal-
  fin printability variant of the unchanged approved baffle; and
- `centered_captive_nut_baffle_finned_variant_bambu_pla.3mf` — high-resolution,
  gasket-face-down printable mesh with generic 0.4 mm-nozzle PLA guidance; and
- `centered_captive_nut_baffle_finned_variant_x2d_pla_matte_project.3mf` —
  native Bambu Studio X2D 0.4 mm / PLA Matte project with the high-quality
  zero-infill process embedded; and
- `centered_captive_nut_assembled.step` — installed-fit reference.

Use `diagnostics.json` for the authoritative installed volume and tuning. The
standard viewers plus the fill, gasket, screw-tunnel, and nut-slot close-section
viewers are generated beside those STEP files.
